# RAO-Bell-Nozzle
Calculation of RAO Bell (Thrust Optimised Parabolic (TOP)) Nozzle \
throat_inlet.m and throat_exit.m calculate throat inlet ve exit curve. \
bell_curve.m calculate nozzle curve \
Example.m is example of how to use \
